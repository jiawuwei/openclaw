# 上传文件按钮模板说明

## 模板要求
`upload_file.png` 模板需要匹配微信对话框中的上传文件按钮（纸夹图标）。

## 截图方法

### 步骤
1. **打开微信桌面客户端**并登录
2. **进入任意对话框**（与好友的聊天窗口）
3. **找到上传文件按钮**：
   - 位置：对话框右下角，输入框右侧
   - 图标：纸夹形状的"文件"图标
   - 通常在"表情"和"截图"按钮之间

4. **截图**：
   - 使用截图工具（Windows: Win+Shift+S）
   - 截取上传文件按钮区域
   - 尺寸：约40×40像素
   - 背景：尽量保持与微信界面一致

5. **保存**：
   - 格式：PNG（支持透明背景）
   - 文件名：`upload_file.png`
   - 位置：技能目录（`skills/wechat-send-file/`）

## 备用模板
如果无法找到准确的微信上传按钮截图，可以使用以下备选方案：

### 方案1：使用通用图标
1. 下载通用文件图标
2. 调整大小到40×40像素
3. 保存为`upload_file.png`

### 方案2：使用现有图片
当前目录中的`wenjian.png`可能是一个上传文件按钮模板，可以尝试：
```bash
# 复制为upload_file.png
copy wenjian.png upload_file.png
```

### 方案3：创建简单模板
使用Paint或其他图片编辑器创建一个简单的纸夹图标：
1. 创建40×40像素的画布
2. 绘制简单的纸夹形状
3. 保存为PNG格式

## 验证模板
创建模板后，可以使用测试脚本验证：
```python
# 测试模板匹配
from PIL import Image, ImageGrab
import pyautogui

# 截屏并显示模板位置
screenshot = ImageGrab.grab()
screenshot.show()

print("请检查截图，上传文件按钮通常在右下角")
```

## 模板优化建议
1. **对比度**：确保图标与背景有足够对比度
2. **尺寸**：与实际微信按钮尺寸相近
3. **颜色**：保留微信图标的默认颜色
4. **透明背景**：使用透明背景以提高匹配精度
