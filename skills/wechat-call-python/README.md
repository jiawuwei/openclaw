# 微信电话自动化技能 (Python版本)

这个技能使用Python自动化控制微信桌面客户端，通过模板匹配技术定位微信界面元素，自动化完成搜索用户和拨打电话的完整流程。

## 特点

- **纯Python实现**: 完全使用Python，移除Node.js依赖
- **跨平台支持**: 支持Windows、macOS和Linux
- **模板匹配**: 使用图像处理技术定位界面元素
- **智能重试**: 支持多次尝试扫描，提高成功率
- **无硬编码坐标**: 基于模板匹配，适应不同屏幕分辨率

## 安装

### 1. 安装Python依赖

```bash
# 进入技能目录
cd skills/wechat-call-python

# 安装基础依赖
pip install -r requirements.txt

# 或者单独安装
pip install pyautogui pillow numpy pyperclip

# 可选: 安装OpenCV以获得更好的模板匹配
pip install opencv-python
```

### 2. 准备模板图片

需要两个模板图片：
- `soupic.png`: 微信搜索图标（放大镜）
- `dianhua.png`: 微信电话图标

可以将这些图片从原Node.js版本的`wechat-call`技能复制过来。

## 使用方法

### 在Moltbot中使用

当用户发送以下格式的消息时，技能会自动激活：
- "用微信给[用户名]打电话"
- "用微信给[用户名]拨打语音"
- "微信电话[用户名]"
- "拨打微信语音[用户名]"

### 命令行测试

```bash
# 模拟测试
python skill_activator.py "用微信给测试用户打电话" --simulate

# 带调试信息
python skill_activator.py "用微信给测试用户打电话" --debug

# 实际运行
python skill_activator.py "用微信给可乐煮鸡打电话"

# 使用自定义置信度
python skill_activator.py "用微信给测试用户打电话" --confidence 0.85
```

### 直接运行Python脚本

```bash
# 帮助信息
python wechat_call.py --help

# 拨打电话
python wechat_call.py --user "用微信给可乐煮鸡打电话"

# 调试模式
python wechat_call.py --user "测试用户" --debug --simulate
```

## 配置

### 命令行参数

- `--user`: 必需，目标用户名或完整消息
- `--debug`: 显示详细调试信息
- `--simulate`: 模拟模式，不执行实际操作
- `--confidence`: 模板匹配置信度阈值 (默认0.9)
- `--use-opencv`: 使用OpenCV进行模板匹配（如果已安装）
- `--no-opencv`: 不使用OpenCV（即使可用）

### 配置文件

可以在Python脚本中修改以下配置：
- `max_retries`: 最大重试次数
- `retry_delay`: 重试延迟（毫秒）
- `search_delay`: 搜索延迟（毫秒）
- `max_scan_attempts`: 最大扫描尝试次数
- `scan_delay`: 扫描间隔延迟（毫秒）

## 技术架构

### 核心模块

1. **pyautogui**: 鼠标键盘自动化控制
2. **PIL (Pillow)**: 图像处理和屏幕截图
3. **模板匹配**: 使用PIL或OpenCV进行图像匹配
4. **智能重试**: 多次尝试机制提高成功率
5. **跨平台支持**: 适配Windows、macOS、Linux

### 工作流程

1. **提取用户名**: 从消息中提取目标用户名
2. **激活微信**: 使用快捷键激活微信窗口
3. **模板匹配搜索**: 查找并点击搜索图标
4. **粘贴搜索**: 粘贴用户名并搜索
5. **模板匹配电话**: 查找并点击电话图标
6. **拨打完成**: 完成电话拨打流程

## 故障排除

### 常见问题

**Q: 无法找到模板匹配**
**A**: 
1. 降低置信度阈值: `--confidence 0.8`
2. 检查模板图片是否正确
3. 确保微信界面已完全加载
4. 确保屏幕分辨率与模板匹配

**Q: 依赖安装失败**
**A**:
1. 确保Python版本 >= 3.8
2. 使用国内镜像: `pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`
3. 检查网络连接

**Q: 剪贴板操作失败**
**A**:
1. 确保已安装pyperclip: `pip install pyperclip`
2. 检查系统剪贴板权限

**Q: 窗口激活失败**
**A**:
1. 确保微信客户端已启动
2. 检查快捷键是否正常工作
3. 尝试手动激活微信窗口

### 调试技巧

1. 使用`--debug`参数查看详细执行信息
2. 使用`--simulate`参数进行模拟测试
3. 检查模板匹配的置信度
4. 查看生成的调试截图

## 性能优化

### 匹配优化

1. **灰度匹配**: 转换为灰度图减少计算量
2. **步进扫描**: 步进扫描提高效率
3. **区域限制**: 限定扫描区域，减少扫描范围
4. **置信度调整**: 适当调整阈值平衡精度和速度

### 内存管理

1. **自动清理**: 及时释放图像内存
2. **临时文件**: 可选择保存临时截图用于调试
3. **错误恢复**: 异常情况下的资源清理
