# 微信电话自动化技能 (Python版本)

这个技能使用Python自动化控制微信桌面客户端，通过**模板匹配**技术定位微信界面元素，自动化完成搜索用户和拨打电话的完整流程。使用纯Python实现，移除所有Node.js依赖。

## 重要特性

1. **纯Python实现** - 完全使用Python，无需Node.js环境
2. **模板匹配** - 使用OpenCV/PIL进行图像模板匹配
3. **跨平台支持** - 支持Windows、macOS和Linux
4. **无坐标计算** - 基于模板匹配定位，无硬编码坐标
5. **智能重试** - 支持多次尝试扫描，提高成功率
6. **模拟人类操作** - 添加随机偏移模拟自然点击
7. **自动提取用户名** - 从消息中自动提取用户名

## 触发条件

当用户消息匹配以下模式时触发：
- "用微信给[用户名]打电话"
- "用微信给[用户名]拨打语音"
- "微信电话[用户名]"
- "打电话给[用户名]"
- "拨打微信语音[用户名]"

## 前提条件

1. **微信桌面客户端** 必须已安装并登录
2. **Python环境** (Python 3.8+)
3. **必要Python包**：
   - pyautogui: 鼠标键盘控制
   - pillow (PIL): 图像处理
   - opencv-python: 模板匹配 (可选，提供更好的匹配)
   - numpy: 数值计算
4. **模板图片** `soupic.png` 和 `dianhua.png` 需正确准备

## 快速安装

### 1. 安装Python依赖
```bash
# 基本依赖
pip install pyautogui pillow numpy

# 可选：OpenCV用于更好的模板匹配
pip install opencv-python

# 或者一次性安装所有依赖
pip install -r requirements.txt
```

### 2. 拨打电话
```bash
# 使用Python脚本
python wechat_call.py --user "用微信给可乐煮鸡打电话"

# 直接用户名格式
python wechat_call.py --user "可乐煮鸡"

# 带调试信息
python wechat_call.py --user "用微信给测试用户打电话" --debug

# 自定义置信度阈值
python wechat_call.py --user "测试用户" --confidence 0.85 --debug
```

## 配置参数

### 命令行参数
- `--user`: **必需**，目标用户名或完整消息
- `--debug`: 可选，显示详细调试信息
- `--simulate`: 可选，模拟模式，不执行实际操作
- `--confidence <数值>`: 可选，模板匹配置信度阈值 (0.0-1.0, 默认0.9)
- `--use-opencv`: 可选，使用OpenCV进行模板匹配（如果安装了OpenCV）

### 匹配参数
- 最大扫描尝试次数：5次
- 扫描间隔延迟：500ms
- 扫描步进：2像素（提高效率）
- 随机偏移范围：±5像素
- 点击延迟：100ms

## 技术实现

### 核心模块
1. **`pyautogui`** - 鼠标键盘自动化控制
2. **`PIL.ImageGrab`** - 屏幕截图
3. **`PIL.Image`** - 图像处理
4. **模板匹配算法** - 使用PIL或OpenCV进行模板匹配
5. **智能重试机制** - 多次尝试，提高成功率

### 工作流程

#### 1. 提取用户名并复制到剪贴板
- 从消息中提取用户名
- 使用`pyperclip`或系统剪贴板API复制用户名

#### 2. 激活微信窗口
- 按 `Win+D` 返回桌面
- 按 `Ctrl+Alt+W` 唤醒微信窗口
- 等待窗口完全加载

#### 3. 模板匹配查找搜索图标
- 截取当前屏幕图像
- 使用`soupic.png`模板进行匹配
- 智能多次扫描，直到找到匹配位置
- 在匹配位置的中心点击（添加随机偏移）

#### 4. 粘贴用户名并搜索
- 从剪贴板粘贴用户名到搜索框
- 按回车键确认并进入对话框
- 等待对话框加载

#### 5. 模板匹配查找电话图标
- 截取当前屏幕图像
- 使用`dianhua.png`模板进行匹配
- 智能多次扫描，直到找到匹配位置
- 在匹配位置的中心点击（添加随机偏移）拨打电话

#### 6. 自动清理桌面截图
- 电话拨打完成后
- 自动搜索桌面上的截图文件
- 删除所有匹配模式的截图文件
- 保持桌面整洁

## 故障排除

### 常见问题

**问题**: 无法找到模板匹配
**解决**: 
1. 降低置信度阈值：`--confidence 0.8`
2. 检查模板图片是否正确
3. 确保微信界面已完全加载
4. 确保截图区域包含目标元素

**问题**: 依赖安装失败
**解决**: 
1. 确保Python版本 >= 3.8
2. 使用国内镜像：`pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`
3. 确保网络连接正常

**问题**: 剪贴板复制失败
**解决**: 
1. 安装pyperclip：`pip install pyperclip`
2. 确保用户名不包含特殊字符

**问题**: 窗口未正确激活
**解决**: 
1. 确保微信客户端已启动
2. 检查快捷键`Ctrl+Alt+W`是否正常工作
3. 尝试手动激活微信窗口
